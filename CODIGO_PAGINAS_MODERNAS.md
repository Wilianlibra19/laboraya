## 📋 Instrucciones para Subir a GitHub

### Opción 1: Interfaz Web (Recomendado)

1. **Ve a tu repositorio:**
   ```
   https://github.com/WilliamRiv19/boletape-privacy
   ```

2. **Click en "Add file" → "Upload files"**

3. **Arrastra los 3 archivos de la carpeta `docs/`:**
   - `index.html`
   - `terminos.html`
   - `privacidad.html`

4. **Escribe un mensaje de commit:**
   ```
   Agregar páginas legales de LaboraYa
   ```

5. **Click "Commit changes"**

---

### Opción 2: GitHub Desktop

1. **Abre GitHub Desktop**

2. **File → Add Local Repository**

3. **Selecciona la carpeta de LaboraYa**

4. **Verás los cambios en la lista**

5. **Escribe un mensaje:**
   ```
   Agregar páginas legales modernas
   ```

6. **Click "Commit to main"**

7. **Click "Push origin"**

---

## 🚀 Activar GitHub Pages

1. **Ve a Settings:**
   ```
   https://github.com/WilliamRiv19/boletape-privacy/settings/pages
   ```

2. **Configura:**
   - Branch: `main`
   - Folder: `/docs`
   - Click **Save**

3. **Espera 2 minutos**

4. **Tus URLs serán:**
   ```
   https://williamriv19.github.io/boletape-privacy/
   https://williamriv19.github.io/boletape-privacy/terminos.html
   https://williamriv19.github.io/boletape-privacy/privacidad.html
   ```
