## 🚀 Cómo Usar en Play Store

### Paso 1: Subir a GitHub
```bash
git add docs/
git commit -m "Agregar páginas legales"
git push
```

### Paso 2: Activar GitHub Pages
1. Settings → Pages
2. Source: main branch
3. Folder: /docs
4. Save

### Paso 3: Obtener URLs
```
https://TU_USUARIO.github.io/laboraya/terminos.html
https://TU_USUARIO.github.io/laboraya/privacidad.html
```

### Paso 4: Pegar en Play Store Console
- Privacy Policy URL: URL de privacidad.html
- Terms of Service URL: URL de terminos.html
